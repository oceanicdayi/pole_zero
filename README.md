# pole_zero

這份檔案是來自 **TW 網（台灣地震觀測網）**，測站代碼 **CHK（成功氣象站）** 的儀器響應檔案（SAC Pole-Zero 格式）。根據檔案內的 Metadata，這是一台 **SMART24A** 紀錄器搭配內建 **PA-2 加速度感測器** 的強震儀 。

以下利用此檔案中的數值，具體說明強震儀的 Pole 與 Zero 物理意義：

### 1. 輸入與輸出單位 (Units)

* 
**Input Unit (輸入單位)**: `M` (Meters，公尺/位移) 。


* 
**Output Unit (輸出單位)**: `COUNTS` (數位計數) 。


* **說明**: 雖然這是一台「強震儀（加速度計）」，但在地震學的儀器響應檔案標準（如 SEED/SAC 格式）中，通常會將「輸入」統一標準化為**地面位移 (Displacement)**。因此，Pole 與 Zero 的配置必須包含將「位移」轉換為「加速度」的數學過程。

### 2. 零點分析 (Zeros)

檔案中列出了 **3 個零點 (Zeros)** ：

* **數值**:
1. `0.0 + 0.0i` (原點)
2. `0.0 + 0.0i` (原點)
3. `-3333.0 + 0.0i` (實數軸負端)


* **物理意義**:
* **原點的兩個零點 ()**: 這對應到物理學上的**微分**。
* 速度是位移的微分 ()。
* 加速度是位移的二次微分 ()。
* 因為輸入定義為「位移 (M)」，而儀器實際感測的是「加速度」，所以這兩個在原點的零點代表了 ** 的物理轉換**。這讓儀器對低頻（靜態）位移的靈敏度為零，但能正確反映加速度。


* **第三個零點 (-3333)**:這是一個位於極高頻（約 530 Hz）的零點。這通常用於高頻響應的微調或補償，位於地震主要觀測頻帶之外。



### 3. 極點分析 (Poles)

檔案中列出了 **4 個極點 (Poles)** ：

* **複數極點對**: `-742.0 ± 1014.0i`
* **實數極點**: `-866.3`
* **實數極點**: `-5638.0`

**關鍵解析：複數極點對 (-742 ± 1014i)**
這是決定儀器高頻特性的核心部分。我們可以計算其自然頻率（Corner Frequency）：

1. **角頻率 ()**: 根據畢氏定理 。
2. **頻率 ()**: 。

* **物理意義**:
* 這台強震儀系統的轉角頻率約為 **200 Hz**。
* 相較於寬頻地震儀（通常轉角在 0.008 Hz 或 120秒），強震儀是為了記錄高頻猛烈的震動。
* 在這個頻率（200 Hz）以下，儀器對「加速度」的響應是平坦的（Flat response to acceleration）。
* 其他的實數極點（-866 和 -5638）則提供了額外的高頻濾波，這通常與**抗混疊濾波器 (Anti-aliasing filter)** 或電子電路的頻寬限制有關，確保高頻雜訊不會干擾訊號。



### 4. 靈敏度與增益 (Constant/Sensitivity)

* 
**A0 (Normalization Factor)**: `2.313560e+09` 。


* 
**Sensitivity**: `3.121640e+05` (M/S**2) 。


* 
**CONSTANT**: `7.222102e+14` 。


* **說明**:
* 系統總增益（CONSTANT）是 A0 與 Sensitivity 的乘積。
* 當我們用程式去除儀器響應時，會用這個常數將儀器記錄到的 `COUNTS` 還原回物理單位（在此例中，若除以整個 Transfer Function，會還原回位移 M；若只除以靈敏度，則得到加速度）。



### 總結

這個檔案描述了一個標準的**強震觀測系統**：

1. 透過 **2 個原點零點** 將輸入的位移轉換為加速度。
2. 擁有極寬的頻寬，其高頻轉折點（Corner Frequency）高達 **200 Hz**，遠高於一般地震分析感興趣的頻率（通常 < 50 Hz），這保證了在記錄強烈地震波時，儀器不會產生失真或過早衰減。
