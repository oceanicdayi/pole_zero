# Example Skill

這是一個示例 skill，用於展示 Antigravity skills 的基本結構和用法。

## 何時使用此 skill

- 當需要了解如何創建新的 skill 時
- 當需要參考 skill 的標準格式時
- 當需要學習 skill 的最佳實踐時

## Skill 的基本結構

每個 skill 都應該包含：

1. **YAML frontmatter**: 包含 `name` 和 `description`
2. **清晰的標題和說明**: 解釋 skill 的用途
3. **使用時機**: 明確說明何時應該使用此 skill
4. **具體指引**: 提供詳細的步驟或規範

## 最佳實踐

- 保持 skill 專注於單一任務
- 使用清晰、具體的描述
- 提供實際可執行的指引
- 包含範例和參考資料（如需要）

## 範例用法

當 agent 看到與此 skill 相關的任務時，會自動讀取並遵循這些指引。
